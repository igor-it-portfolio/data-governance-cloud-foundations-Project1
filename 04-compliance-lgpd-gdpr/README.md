# Compliance LGPD e GDPR

Este diretório contém materiais relacionados à conformidade legal com LGPD e GDPR.

Inclui:
- Checklist de conformidade
- Avaliação de riscos
- Controles mínimos exigidos
- Boas práticas de auditoria e evidências

O objetivo é garantir que o tratamento de dados pessoais esteja em conformidade com a legislação aplicável.
