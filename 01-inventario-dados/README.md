# Inventário de Dados

Este diretório contém o inventário de dados da empresa fictícia, elaborado como base para a Governança de Dados.

O inventário documenta:
- Tipos de dados tratados
- Classificação da informação
- Área responsável
- Papéis de governança (Owner, Steward, Admin)
- Riscos associados
- Ciclo de vida dos dados

Este material é a base para políticas, compliance e controles de acesso.
