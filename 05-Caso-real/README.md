# Caso Real Simulado — Governança de Dados na Prática

Este diretório apresenta um caso real simulado, demonstrando como a Governança de Dados resolve problemas concretos de uma empresa.

O caso aborda:
- Coleta de dados pessoais de clientes via site
- Armazenamento em ambiente cloud
- Riscos de vazamento e acesso indevido
- Aplicação de governança para mitigação de riscos
- Adequação à LGPD e GDPR

O objetivo é demonstrar aplicação prática da governança em um cenário realista.
