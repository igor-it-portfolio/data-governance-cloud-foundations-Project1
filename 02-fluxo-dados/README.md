# Fluxo de Dados e Ciclo de Vida

Este diretório contém o fluxograma do ciclo de vida dos dados, modelado no Draw.io.

O fluxo representa:
- Coleta dos dados
- Uso interno
- Armazenamento
- Compartilhamento
- Retenção
- Descarte ou anonimização

Cada etapa indica responsabilidades, riscos e controles de governança.
