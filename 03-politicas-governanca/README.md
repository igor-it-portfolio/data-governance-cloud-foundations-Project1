# Políticas de Governança de Dados

Este diretório reúne as políticas de Governança de Dados da empresa fictícia.

Inclui:
- Política de classificação da informação
- Política de controle de acesso
- Definição do ciclo de vida dos dados
- Papéis e responsabilidades

As políticas foram elaboradas considerando boas práticas de mercado e conformidade com LGPD e GDPR.
